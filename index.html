Installing Experimental Packages

County Linux is based on Debian Stable and allows you to install packages directly from Sid (unstable) or Testing branches for cutting-edge updates.

Debian Sid (Unstable)

Get the latest versions of software and drivers:

sudo apt install package/unstable

Debian Testing

Stay more stable, but still ahead of Stable branch:

sudo apt install package/testing
